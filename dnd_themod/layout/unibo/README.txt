## DOCUMENTAZIONE

All'interno di questa cartella sono presenti i seguenti file e cartelle con le seguenti caratteristiche

(* elementi obbligatori)

Documento INI che fornisce informazioni generali sul tema (quali nome, descrizione, autore ..)

Documento CSS che definisce l'aspetto grafico del LO.
